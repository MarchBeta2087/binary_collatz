# Binary Collatz v0.2

Copyright (c) 2026 MarchBeta2087

一个基于大整数位操作，演示 Collatz 过程的 C 语言程序，对给定的非零二进制串迭代执行删除尾零、删除尾随 "01" 模式、乘以 3 以及变换操作，直至结果变为 `11b`。

v0.2 新增多种输出模式、命令行参数支持，并修复了若干边界情况 bug，同时将输出逻辑解耦为独立模块。

## 算法流程

1. 删除输入二进制串末尾所有的 `0`，得到 Y。
2. 删除 Y 末尾所有的 `01` 模式（如 `100101` → `10`），得到 Z。
3. 将 Z 乘以 3，得到 W。
4. 若 W 恰好为 `11b`，则终止；否则，将 W 中最低的 `0` 变为 `1`，并删除所有更低的有效位，得到新的 Y，返回步骤 2 继续。

## 原理

Z000000b × 3 + (010101b × 3 + 1) = Z000000b × 3 + (111111b + 1) = Z000000b × 3 + 1000000b = (Z × 3 + 1) × 1000000b。需要注意的是，000000b 和 010101b 在这里并非分别是具体的 6 个 0 和 3 个 01，而是**泛指** 2N 个 0 和 N 个 01

## 为什么 W 是 11b 则终止？为什么不是 111b、1111b？

因为 111b 是 7，7 不是 3 的倍数，1111b 是 15，15 = 5 × 3，5 是 101b，但 101b 末尾有 01，在步骤 2 已经被删了。

11b 终止的原理：1 × 3 + 1 = 4，4 ÷ 2 = 2，2 ÷ 2 = 1。

## 新增功能（v0.2）

- **命令行参数**  
  `--mode=console`（默认）、`--mode=silent`、`--mode=file-output` 控制中间信息输出行为。  
  `-o <file>` 配合 `--mode=file-output` 指定输出文件。  
  `-h` / `--help` 显示帮助信息，`-v` / `--version` 显示版本信息。

- **输出模块解耦**  
  输出相关函数独立为 `output.h` / `output.c`，提供 `process_binary` 和 `output_message` 接口，根据模式决定输出目标或静默。

- **输入规范化与错误处理**  
  自动去除输入二进制串的前导零；若输入全为零则直接退出，避免无限循环。

- **Bug 修复**  
  修复 `step4_transform` 在输入为空时 `end` 未更新的缺陷，杜绝因空大整数导致的状态异常。

## 构建与运行

### 依赖

- GCC 或支持 C99 以上的编译器
- 标准 C 库

### 编译

```bash
gcc -std=c99 -O2 -o binary_collatz main.c binary_bigint.c output.c
```

### 使用

从标准输入读取一个仅包含字符 0 和 1 的字符串（可以带前导零，不能全为零），根据命令行选项输出中间步骤。

#### 示例

```bash
# 控制台模式（默认）
echo "100101" | ./binary_collatz

# 静默模式（无中间输出）
echo "100101" | ./binary_collatz --mode=silent

# 输出到文件
echo "100101" | ./binary_collatz --mode=file-output -o result.txt

# 查看帮助
./binary_collatz -h

# 查看版本
./binary_collatz -v
```

## 内部实现

- 使用动态数组以小端序存储大整数，每个元素为 `uint64_t`。
- 支持从 512-bit 开始自动扩容至任意长度（理论上限达 2^63 bit）。
- 采用“软删除”策略，通过调整 `start` 和 `end` 指针避免不必要的数据搬移，仅在必要时触发 `homing` 和扩容。
- 输出控制模块通过函数指针/条件分支实现多种输出策略，保持算法代码与 I/O 逻辑的松耦合。

## 文件结构

- `binary_bigint.h` : 二进制大整数结构体定义、基础操作与算法步骤声明。
- `binary_bigint.c` : 大整数初始化、扩容、移位、乘 3、变换等具体实现。
- `output.h` / `output.c` : 模式感知的输出封装，提供 `process_binary` 和 `output_message`。
- `main.c` : 命令行解析、输入读取、算法主循环及资源管理。
- `LICENSE.txt` : MIT 许可证文件。
